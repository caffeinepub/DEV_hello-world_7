# Design Brief

## Direction

Utilitarian Clarity — a minimal hello world interface that lets typography carry all meaning.

## Tone

Brutally minimal, refined restraint — no animations, no gradients, no decorative elements.

## Differentiation

Pure content focus: the text is the entire interface, centered and dignified.

## Color Palette

| Token      | OKLCH       | Role                    |
| ---------- | ----------- | ----------------------- |
| background | 0.99 0 0    | Light, neutral backdrop |
| foreground | 0.15 0 0    | Dark text               |
| primary    | 0.15 0 0    | Interactive elements    |
| muted      | 0.95 0 0    | Secondary backgrounds   |

## Typography

- Display: DM Sans — heading and body text
- Body: DM Sans — all content
- Scale: hero 6xl bold, body lg regular

## Elevation & Depth

Flat surfaces with no shadows; clarity through spacing and typography only.

## Structural Zones

| Zone    | Background  | Border | Notes         |
| ------- | ----------- | ------ | ------------- |
| Content | background  | —      | Centered page |

## Spacing & Rhythm

Amp spacing: 4rem gaps, centered alignment, breathing room around text.

## Component Patterns

- Text: large, readable, single-column centered

## Motion

- Entrance: none
- Hover: none
- Decorative: none

## Constraints

- No animations
- No decorative elements
- No gradients or shadows
- Typography-only design

## Signature Detail

Contentless restraint: the absence of decoration becomes the design statement.
